# 📚 TIL (Today I Learned)

> 매일 배우고, 기록하고, 설명할 수 있는 개발자가 되기 위한 학습 저장소

안녕하세요.  
백엔드 개발자로서 기본기를 다지며 지속적으로 역량을 확장하고 있습니다.

이 저장소는 단순한 공부 기록이 아니라,  
**이해한 내용을 제 언어로 다시 설명하는 공간**입니다.

---

## 🎯 학습 목표

- CS 기초 체력 강화
- 코딩테스트 문제 해결 능력 향상
- Java 심화 (Lambda, Stream 중심)
- 면접에서 “설명 가능한 수준”까지 정리

## 📂 Repository Structure
```
TIL/
├── algorithm/ # 코딩테스트 & 문제 풀이

```

## 🧠 Study Rules

- 단순 복붙 ❌ → 이해 후 재작성 ⭕
- “면접에서 설명할 수 있는가?” 기준으로 작성
- 시간복잡도 / 동작 원리 반드시 포함
- 하루 1개 이상 기록 유지
